Niet doen

  • Geen style="font-size:13px" — gebruik var(--fs-sm)
  • Geen hardcoded kleuren — alleen via tokens.css
  • Geen ander font dan Geist (op deze tokens)
  • Geen border-radius 6px, 10px, etc. — alleen 4 / 8 / 12 / 50% / 999px
  • Geen spacing 5, 7, 11, 18px — afronden naar 4-grid
  • Geen eigen tokens.css per app — gebruik shared
